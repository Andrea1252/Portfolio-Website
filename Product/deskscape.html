Deskscape

Handcrafted desktop organizer built to study form and order

Deskscape is a hand built desktop organizer created in RISD Design Principles I. With only hand tools, the project focuses on proportion, spacing, and simple geometric structure to turn everyday clutter into a clear layout. The result is a quiet object that demonstrates foundational design thinking through restraint, craft, and functional clarity.
